# Smart To-Do List Application

This repository contains the files for the Smart To-Do List application.

## Features
- Add and remove tasks
- Prioritize tasks
- Responsive design

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mummadimaheswar/aismartirrigationwithweatherapi.git
   ```
2. Open `index.html` in your browser.

## Usage
- Type your task in the input field and click 'Add Task' to add it to the list.
- Click the delete button to remove a task from the list.
